Getting Started

Make sure you have Nodejs and NPM installed on your computer.

NodeJS download link

Run in the Terminal

Use terminal and navigate to project root.

Then run npm install

Then run npm start

Build

Run npm run build to build the project. The build artifacts will be stored in the build/ directory.

Navbar Configure

Step : 1

Got to App.js component, then you found this code. If you chose home page version two. Then follow the setup. src/App.jsx

    
                            
                                    <Switch>
                                    <Route
                                        path="/home-three"
                                        component={HomeThree}
                                    />
                                    <Route
                                        path="/home-two"
                                        component={HomeTwo}
                                    />

                                    <Route
                                        path="/"
                                        component={HomeOne}
                                    />
                                </Switch>

Step : 2

Then you should have remove <Route/> other version one and version three and for version two you have to edit path="/home-two" to path="/" and add exactly . Like the below code. 

    
                        
                            <Switch>
                            <Route
                                path="/home-three"
                                component={HomeThree}
                            />
                            <Route
                                path="/"
                                component={HomeTwo}
                            />
                        </Switch>
